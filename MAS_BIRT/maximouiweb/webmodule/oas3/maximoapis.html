MAXIMO API Categories



Maximo APIs are grouping in a more logical manner, you can click the tag in category to see the details for the APIs.



Title Description Tag Action Enabled
Account Management  APIs around users, groups, security, access control, etc ACCOUNT 
Asset Management  APIs around assets, items, inventory, etc ASSET 
Work Management  APIs around work orders, scheduling, pm, etc WORKMAN 
System Management  APIs around managing system properties, log levels, maxvars, domains etc SYSTEM 
Supply Chain  Purchasing, Contracts SUPPLYCHAIN
